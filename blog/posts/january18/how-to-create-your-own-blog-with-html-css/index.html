How to create a Blog with HTML and CSS

Creating a blogging site using only HTML and CSS seems quite daunting and intimidating, but it was a common approach to maintain a blog before wordpress came to the rescue.

As a front end web developer, I wanted to have a blog, but due my lack of knowledge in php, wordpress and databases, I could not make one. I had two options, either to learn whatever is required to make a blogging site, or use my front end skills to create a blogging site with HTML and CSS. I decided to try the later and this site is the result of all that.

Features of this blog site

  • Posting : Write your post in HTML, update the the entry in a seperate javascript file and thats it! The site will itself add an entry on your blog page and add an entry into the article data table.
  • PWA : This site have features of Progressive Web App, hence user can now read your articles offline and can even add a shortcut on their homescreen as an app.
